# Sample Matter Guide

The included sample matter is synthetic demonstration data only.

It is a fictional workplace dispute scenario involving:

- Multiple documents
- A timeline with supported and partially supported events
- Conflicting recollections
- Policy references
- Correspondence
- A draft demand-style letter
- Missing information
- Unsupported assertions

Every sample matter file includes this note:

> This is synthetic sample data for demonstration purposes only.

## How To Read Outputs

- Start with `chronology.md` for the ordered fact timeline.
- Read `evidence_index.md` to understand document sources and hashes.
- Read `issue_map.md` for issue/risk signals requiring human review.
- Read `missing_info_checklist.md` for intake gaps.
- Read `drafting_support_packet.md` before revising any draft.
- Read `red_team_review.md` and `unsupported_claims_report.md` before sending or relying on draft language.

The harness does not decide legal meaning. It organizes the material for review.
