# Synthetic Sample Matter

This is synthetic sample data for demonstration purposes only.

This folder contains a fictional workplace dispute scenario for testing Lex 4i. It does not describe real people, real companies, real claims, or real legal documents.

## Scenario

Jordan Ellis, a fictional inventory coordinator at Riverbend Components LLC, raised concerns about off-clock work entries and later received a termination notice. The sample matter includes correspondence, notes, a policy excerpt, a witness note, and a draft demand-style letter with intentional support gaps.

The harness uses these files to demonstrate chronology generation, evidence indexing, issue/risk mapping, missing information detection, drafting support, red-team review, and receipt generation.
